# Skill Creator - 技能包创建向导

## 这是什么？

Skill Creator 是一个帮你创建 Claude Code Skill（技能包）的工具。

**通俗来说**：你想让AI学会一个新本事（比如"自动写周报"、"代码审查"、"生成测试用例"），这个工具就像一个老师，手把手教你把这个"新本事"打包成一个标准的技能包，装到AI身上。

## 快速开始

### 方法一：直接对话

对AI说以下任意一句：

- "帮我创建一个skill"
- "我想做一个能自动写周报的技能包"
- "创建一个代码审查的skill"

AI会自动进入创建向导模式，一步步引导你完成。

### 方法二：使用斜杠命令

在Claude Code中输入：

```
/skill-creator
```

## 创建流程概览

```
第1步：告诉AI你想做什么skill
    ↓
第2步：AI会问你几个问题（目的、输入、输出等）
    ↓
第3步：AI帮你选择合适的模板（基础型/工具型/知识型）
    ↓
第4步：AI逐个生成每个文件
    ↓
第5步：AI检查质量
    ↓
第6步：安装到 ~/.claude/skills/ 目录
    ↓
完成！你的新skill可以使用了
```

## 三种模板类型

| 类型 | 适用场景 | 复杂度 | 示例 |
|------|---------|--------|------|
| 基础型 | 纯指令和知识 | 低 | 代码风格检查、文案润色 |
| 工具型 | 需要执行脚本 | 中 | 自动部署、数据处理 |
| 知识型 | 包含大量参考资料 | 中 | 框架最佳实践、API参考 |

## 安装位置

所有skill都安装在：

```
~/.claude/skills/你的skill名/
```

## 文件结构说明

```
你的skill名/
├── skill.md          ← 必需！AI的操作手册
├── README.md         ← 可选。使用说明
├── templates/        ← 可选。模板文件
├── knowledge/        ← 可选。知识库
└── tools/            ← 可选。可执行脚本
```

## 常见问题

**Q: 创建一个skill需要多长时间？**
A: 简单的基础型skill几分钟就能完成，复杂的工具型可能需要更多时间来调试脚本。

**Q: 我不会写代码，能创建skill吗？**
A: 可以！基础型skill只需要用自然语言描述就行，不需要写任何代码。

**Q: 创建好的skill怎么分享给别人？**
A: 把整个skill文件夹复制给对方，对方放到自己的 `~/.claude/skills/` 目录下即可。

**Q: skill可以修改吗？**
A: 当然可以。直接编辑 `skill.md` 或其他文件即可，修改立即生效。

**Q: 怎么删除一个skill？**
A: 删除 `~/.claude/skills/你的skill名/` 整个文件夹即可。
